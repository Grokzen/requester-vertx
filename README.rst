===============
requester-vertx
===============

Webbapp that can make simple requests and some admin functionality. Created with vert.x and knockout.js



System dependencies
===================

#. vert.x >= 2.1M1
#. MongoDB



How to run
==========

#. Run: vertx run app.py
#. Open webbbrowser and navigate to http://localhost:8080



Functionality
=============

#. Add task
#. Remove task
#. Save task
